# 🐠 KOKI FARM - Sistem Manajemen Usaha Ikan Koki

Sistem informasi berbasis web untuk mengelola usaha budidaya dan penjualan ikan koki.

## 📋 Fitur Utama

| Modul | Deskripsi |
|-------|-----------|
| 📊 Dashboard | Statistik real-time stok, penjualan, kematian |
| 🏊 Manajemen Kolam | CRUD kolam + riwayat aktivitas |
| 🐟 Data Ikan | Tracking per individu/batch dengan grading |
| 🛒 Pembelian | Record pembelian benih dari supplier |
| 💰 Penjualan | Transaksi penjualan + invoice PDF |
| 📦 Pengiriman | Tracking resi & status pengiriman |
| 👥 Customer & Supplier | Database pembeli dan supplier |
| 🏥 Kesehatan | Log kesehatan & catatan kematian |
| 🍽️ Pakan | Stok pakan & jadwal pemberian |
| 📈 Laporan | Laporan stok, penjualan, kematian |
